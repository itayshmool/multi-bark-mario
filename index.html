WORLD 2-1

POWER-UPS & FEATURES

? 🤖
MULTI-BACKEND

Run Claude Code, Cursor, OpenAI Codex, or Google Gemini. Tag #claude-code or #gemini to pick per-message. Switch anytime.

? 💾
SAVE STATE

Persistent sessions survive server restarts. Pups remember their conversation history. /reborn to resurrect shelved pups with full context intact.

? 🎙️
VOICE CONTROL

Send voice messages — whisper.cpp transcribes locally. Supports 90+ languages. No cloud upload. ffmpeg converts audio on your machine.

? ♥
EXTRA LIVES

Automatic fallback: retry → reset → switch backend. Context is preserved via rolling summaries. Agents recover from crashes, rate limits, and context overflows.

? 📦
ITEM BOX

Pups deliver files directly to your chat. Drop anything into the outbox — images, diffs, zips — and the server sends it back via sendFile().

? 🤝
CO-OP MODE

Pups spawn sub-agents with bark delegate. Add --branch for isolated branch + PR. Up to 3 sub-agents per pup. Max depth: 1.

WORLD 3-1

CHEAT CODES

★ ENTER THESE IN ANY CHAT ★

/status Refresh the pinned status message showing all active pups
/create Reply to any message to spawn a new pup with that context
/stop Send Ctrl+C to a running pup or use /stop pack for all
/clear Shelve a pup (soft-delete). History preserved for /reborn
/reborn Resurrect a shelved pup with its full session history intact
/reset Wipe pup memory. Stays active, same backend, new session
/daily Request a one-line standup from every active pup at once
/stats Show usage and cost summary. /stats name for one pup
/backends Show available LLM backends and their capabilities
/skills Show available skills. /skill name @pup to add to a pup
/delete Permanently remove a pup. Frees the name for reuse
/restart Restart the server. Auto-restarts via start.sh wrapper
USE PACK INSTEAD OF NAME FOR BULK OPS · @NAME MSG TO ROUTE DIRECTLY TO A PUP

WORLD 4-1

SETUP & INSTALL

🪙 1
CLONE THE REPO

Grab the source from GitHub and jump into the directory.

git clone https://github.com/itayshmool/multi-bark-pack cd multi-bark-pack
🪙 2
RUN PREREQUISITES

Install Homebrew, Node, yarn, tmux, Claude CLI, ffmpeg, and whisper.cpp.

bash prerequisites.sh # installs: brew, node, yarn, tmux, # claude, ffmpeg, whisper.cpp
🪙 3
CONFIGURE .ENV

Set your chat platform tokens and enable your preferred adapters.

# .env DEFAULT_BACKEND=claude-code WA_ENABLED=true WA_GROUP=bark-pack TELEGRAM_TOKEN=your-bot-token TG_OWNER=your-telegram-id SLACK_BOT_TOKEN=xoxb-... SLACK_APP_TOKEN=xapp-...
🪙 4
START THE SERVER

Launch with auto-restart. Send a message to spawn your first pup.

yarn start # server starts + auto-restarts on clean exit # send a msg to spawn Mario, Luigi, or Peach # tmux attach -t bark-Mario to watch
